## What is this?

This repository is my personal CS study journal.

I come from an IT and web development background, but I realized I was missing the foundation —
the real understanding of how computers actually work under the hood.
So I decided to start from zero and learn CS the hard way, from bits and bytes up.

Every topic has two things:

- A **README** with my notes and explanations in plain human language
- A **reference.c** file with working C code that demonstrates the concept

---

## Why C?

C forces you to think at the machine level. No garbage collection, no hidden abstractions,
no hand-holding. If you understand C, you understand what every other language is doing
underneath its surface.
